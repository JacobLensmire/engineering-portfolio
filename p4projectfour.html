Review of Documentation

When reviewing the e-NABLE hub and the NIH 3D Print Exchange, both resources provide essentially identical information, including the same assembly videos and instruction mats. The primary difference is found when downloading the actual files, as e-NABLE allows the direct download of STL files for each part while NIH 3D sends you to Thingiverse to download the files, but this extra step does not create added confusion. First Print

For our first print, we decided to print all of the smaller pieces in a single print, and the larger ones on their own. We used a base for everything except the palm due to adhession issues, and used supports for the palm. After printing, we found that two pieces, the palm and the proximals, failed (unfortunately, these were discarded before being recorded). It was found that when using supports with the palm, the printer would become misaligned and would fail. The proximals, however, failed due to the base not actually providing adhesion, but rather printing around the too-high parts.

Second Print

For the second print, we were able to use all parts from the previous aside from the previously mentioned proximals and palm. For the palm, we simply removed the supports and it succesffuly printed. For the proximals, we used a 2 layer raft to provide adequate adhesion. This print was succesful.
